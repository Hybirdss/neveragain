# NeverAgain - Product Requirements Document

## 비전

> **일반인도 매일 들어와서 보는 지진 시뮬레이션 도구**

JMA EEW(긴급지진속보) 파이프라인을 웹에서 재현하여, 일본에서 발생하는 지진의 진도 분포를 실시간으로 시각화한다. Google Earth급 3D 글로브 위에 등진도선, 파동 전파, 시나리오 시뮬레이션을 얹어 전문 지식 없이도 직관적으로 지진 영향을 이해할 수 있게 한다.

이 도구는 단순한 지진 정보 뷰어가 아니다. 지진공학 엔진(GMPE)이 코어이며, 시각화는 그 출력 위에 얹는 레이어다. 매일 방문하고 싶은 경험을 제공하는 것이 핵심이다.

## 대상 사용자

| 사용자 유형 | 설명 | 핵심 니즈 |
|------------|------|----------|
| **일반인** (Daily Visitors) | 지진에 관심 있는 일반 사용자. 매일 방문하여 최근 지진 현황을 확인한다. | 직관적 UI, 빠른 로딩, 시각적 임팩트 |
| **지진 연구자** | 지진 데이터를 분석하고 시나리오를 시뮬레이션하는 연구자. | GMPE 정확도, 시나리오 커스터마이징, 데이터 내보내기 |
| **방재 교육자** | 학교/기관에서 지진 교육에 활용하는 교육 담당자. | 타임라인 재생, 역사적 지진 리플레이, 난카이 시나리오 |

## 핵심 기능

### 1. GMPE Engine (지진동 예측 엔진)

**개요**: Si & Midorikawa (1999) Ground Motion Prediction Equation을 구현하여, 지진 파라미터(규모, 깊이, 위치)로부터 일본 전역의 진도 분포를 계산한다.

**상세 요구사항**:
- Si & Midorikawa (1999) GMPE 논문의 PGV(최대지반속도) 예측식 구현
- JMA EEW 파이프라인 재현: 규모 → PGV 그리드 → JMA 진도 변환
- 단층 유형(crustal, interplate, intraplate)에 따른 보정 계수 적용
- 이상거리(hypocentral distance) 기반 감쇠 모델
- Web Worker에서 그리드 계산을 수행하여 메인 스레드 블로킹 방지
- 계산 결과를 d3-contour로 등진도선(isoseismal contour)으로 변환

**검증 기준**: 역사적 지진(1995 고베, 2011 토호쿠, 2016 쿠마모토 등) 실측 진도와 비교하여 JMA 진도 기준 ±1.0 이내 정확도.

### 2. 3D Globe (Google Earth급 3D 글로브)

**개요**: globe.gl + Three.js 기반의 인터랙티브 3D 글로브. 어두운 지구 텍스처 위에 지진 데이터를 시각적으로 표현한다.

**상세 요구사항**:
- globe.gl 기반 3D 글로브 렌더링
- Earth night texture로 어두운 분위기 연출
- 구조판 경계(tectonic plate boundaries) GeoJSON 오버레이
- 지진 발생 위치에 펄스 마커 표시 (규모에 비례하는 크기)
- 등진도선을 글로브 표면에 polygon layer로 투영
- 파동 전파 동심원 애니메이션 (P파 파란색, S파 빨간색)
- 마우스/터치 인터랙션: 회전, 줌, 패닝
- 지진 클릭 시 자동 줌인 카메라 연출

**성능 기준**: 60fps 유지. 등진도선 + 마커 + 파동 애니메이션 동시 렌더링 시에도 프레임 드랍 없음.

### 3. Real-time USGS Feed (실시간 지진 데이터)

**개요**: USGS Earthquake API를 60초 간격으로 폴링하여 일본 주변 지진 데이터를 실시간으로 가져온다.

**상세 요구사항**:
- USGS FDSN Event API (`earthquake.usgs.gov/fdsnws/event/1/query`) 연동
- 60초 주기 폴링 (configurable)
- Japan bounding box 필터: 위도 24~46, 경도 122~150
- 최소 규모 필터: M2.5 이상 (configurable)
- 중복 이벤트 제거 (event ID 기반)
- 새 지진 감지 시 자동 GMPE 계산 트리거
- 네트워크 오류 시 exponential backoff 재시도
- 오프라인 시 마지막 데이터 유지

**데이터 포맷**: GeoJSON Feature Collection (USGS 표준)

### 4. Timeline Player (타임라인 플레이어)

**개요**: 시간축을 따라 지진 이벤트를 재생하는 플레이어. 역사적 지진 리플레이와 실시간 모드를 지원한다.

**상세 요구사항**:
- Play / Pause / Stop 컨트롤
- 재생 속도 조절: 1x, 10x, 100x, 1000x
- 시간 범위 선택: 최근 24시간, 7일, 30일, 사용자 지정
- 타임라인 바에 지진 이벤트 틱 마크 표시
- 현재 시점의 지진만 글로브에 표시
- 역사적 지진 시퀀스 리플레이 (예: 2016 쿠마모토 본진-여진 시퀀스)
- 실시간 모드로 전환 시 자동으로 최신 데이터 따라감

### 5. Scenario Simulation (시나리오 시뮬레이션)

**개요**: 사전 정의된 시나리오 또는 사용자 클릭으로 가상 지진을 생성하고 시뮬레이션한다.

**상세 요구사항**:
- **난카이 트러프 시나리오**: 다중 세그먼트 동시 파열, 순차 파동 전파, 전체 시뮬레이션 30초 이내
- **클릭 시뮬레이션**: 글로브 아무 곳이나 클릭하여 가상 지진 생성
  - 클릭 위치 = 진앙
  - 규모, 깊이 슬라이더로 파라미터 조정
  - 즉시 GMPE 계산 및 시각화
- **프리셋 시나리오**: 난카이 트러프, 수도직하, 일본해구 등 주요 시나리오 내장
- 시나리오 결과에 피해 예상 요약 표시 (진도 분포 통계)

## 범위 정의

### In Scope (구현 범위)

- Si & Midorikawa (1999) GMPE 기반 진도 예측
- JMA 진도 계급 (0, 1, 2, 3, 4, 5-, 5+, 6-, 6+, 7) 시각화
- 3D 글로브 기반 시각화 (globe.gl + Three.js)
- USGS 실시간 지진 데이터 연동
- 타임라인 기반 이벤트 재생
- 난카이 트러프 등 시나리오 시뮬레이션
- 클릭 기반 가상 지진 시뮬레이션
- 반응형 웹 (데스크톱 우선, 태블릿 지원)
- 다크 테마 UI
- P파/S파 전파 애니메이션
- 구조판 경계 표시

### Out of Scope (구현 범위 외)

- 모바일 네이티브 앱
- 사용자 인증 / 로그인
- 지진 데이터 저장 서버 (클라이언트 전용)
- 다국어 지원 (1차 버전은 일본어/영어 혼용 UI)
- 쓰나미 시뮬레이션
- 건물 피해 상세 모델링
- Site amplification (지반 증폭) 상세 모델
- 과거 지진 데이터 대량 다운로드
- API 서버 구축

## Non-Goals (명시적 비목표)

이 프로젝트는 다음을 의도적으로 목표로 하지 않는다:

1. **실시간 조기경보 시스템이 아니다**: 이 도구는 교육 및 시각화 목적이다. 실제 지진 발생 시 대피 판단의 근거로 사용해서는 안 된다. 실시간 데이터에는 USGS API의 수십 초~수 분 지연이 포함된다.

2. **연구 등급 지진재해 평가 도구가 아니다**: GMPE 구현은 Si & Midorikawa (1999) 단일 모델에 기반하며, 지반 조건, site effect, 비선형 응답 등을 고려하지 않는다. 학술 논문이나 정책 결정의 근거로 직접 사용하기에는 부족하다.

3. **범용 지진 데이터 플랫폼이 아니다**: 일본 주변 지진에 특화되어 있으며, 전 세계 지진을 대상으로 하지 않는다.

## 용어 정의

| 용어 | 설명 |
|------|------|
| **GMPE** | Ground Motion Prediction Equation. 지진동 예측식. 지진 파라미터로부터 특정 지점의 지반 진동 강도를 예측하는 수식 |
| **JMA 진도** | 일본 기상청 진도 계급. 0~7 (5와 6은 강/약으로 세분, 총 10단계) |
| **PGV** | Peak Ground Velocity. 최대지반속도. 지반이 흔들리는 최대 속도 (cm/s) |
| **PGA** | Peak Ground Acceleration. 최대지반가속도. 지반이 흔들리는 최대 가속도 (gal) |
| **등진도선** | Isoseismal contour. 같은 진도를 가진 지점을 연결한 곡선 |
| **난카이 트러프** | Nankai Trough. 일본 남쪽 해구. M8~9급 거대지진 발생 가능 지역 |
| **EEW** | Earthquake Early Warning. 긴급지진속보 |
| **이상거리** | Hypocentral distance. 진원(지하)에서 관측점까지의 직선 거리 |
